# PlataformaDeskHub
Plataforma DeskHub

DeskHub es una plataforma de gestión de espacios de trabajo híbridos propuesta para optimizar la reserva y asignación de puestos de trabajo en organizaciones que implementan modelos de trabajo flexible para sus colaboradores, mientras los administradores mantienen control y visibilidad sobre la ocupación de las oficinas.
